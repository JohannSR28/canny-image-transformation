# Canny Image Transformation

Live: https://johannsr28.github.io/canny-image-transformation/

Upload an image and instantly generate a **Canny edge-detected** (black & white) version.

✔ 100% client-side  
✔ No backend, no uploads  
✔ Runs entirely in your browser  

Built with **HTML, TailwindCSS, and OpenCV.js**.
